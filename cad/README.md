# Irish Flute CAD Starter

No CAD solids are included yet. Build the first model from the measured D4 prototype, not only from the first-order table.

## Suggested Parameters

- `bore_id = 0.750 in`
- `body_od = 1.000 in`
- `body_length = measured after V-FUND`
- `hole_positions = measured/tuned values from validation.csv`
- `embouchure_window = measured from successful scrap and prototype cuts`

## Model Order

1. Revolved or extruded cylindrical body using bore axis as origin.
2. Embouchure cut feature with editable bevel.
3. Six tone-hole cuts driven by a design table.
4. Optional ergonomic exterior shaping.
