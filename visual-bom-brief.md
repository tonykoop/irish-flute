# Irish Flute Visual BOM Brief

## Layout

Create a one-page visual BOM after materials are sourced.

Required panels:

- Full blank with grain direction and marked bore axis
- Long boring bit and reamer
- Embouchure tool set
- Tone-hole bit set
- Bore finish and exterior finish
- Completed flute profile after first build

## Image Requirements

- Neutral background, top-down for parts, 3/4 view for finished body.
- Include a ruler or calipers in process shots where scale matters.
- Label estimated dimensions as "first-order" until `validation.csv` contains measurements.
