# learn-to-play / Irish Flute (D)

Starter songbook for the **6-hole D Irish flute** from the [`irish-flute`](https://github.com/tonykoop/irish-flute) build packet. Generated by the [`sheet-music`](https://github.com/tonykoop/sheet-music) skill.

The Irish flute is a simple-system wooden flute in D — similar to a tin whistle in fingering but with a more breathy, open tone and slower response.

Range: **D5 – D7** (two octaves). Key of D major / D mixolydian.
